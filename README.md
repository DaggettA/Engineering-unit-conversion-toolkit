# Engineering Unit Conversion Toolkit

## Overview

This project is a beginner engineering toolkit focused on dimensional analysis and unit conversions used in aviation, mechanical, and aerospace problem solving.

The goal is to practice engineering fundamentals, programming, and technical documentation while building a useful tool for common unit conversions.

## Why This Project Matters

Engineering problems depend on correct units. A wrong conversion can lead to incorrect calculations, bad assumptions, or unsafe results. This project helps reinforce dimensional analysis and unit consistency.

## Current Features

- Feet to meters
- Pounds to kilograms
- Knots to miles per hour
- Horsepower to watts
- Fahrenheit to Celsius

## Engineering Concepts Used

- Dimensional analysis
- Conversion factors
- Unit cancellation
- Engineering notation
- Basic programming functions

## Tools Used

- Python
- GitHub
- Markdown
